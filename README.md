## Steps

### Clone the project
`git clone https://github.com/Picus-Security-Golang-Backend-Bootcamp/homework-1-oguzhantasimaz.git`

### To run the project
`cd homework-1-oguzhantasimaz/cmd/homework-1-oguzhantasimaz`

`go run . search <Book Name>`

`go run . list`
